API Integration

Integrate TeachMate AI's powerful course features into your platform

API Endpoint

https://teachmateai.com/api/v1/courses

Base URL for all course-related API endpoints. Use this URL with the appropriate HTTP method and parameters.

Authentication

Authorization: Bearer YOUR_API_KEY

Add this header to all API requests. Replace YOUR_API_KEY with your actual API key.

Course Widget Integration

<script src="https://teachmateai.com/widget/course.js"></script> <div id="teachmate-course-widget" data-course-id="YOUR_COURSE_ID"></div> <script> TeachMateCourse.init({ theme: 'light', language: 'en', autoSync: true }); </script>

Add this code to your HTML where you want the course widget to appear. Replace YOUR_COURSE_ID with your actual course ID.

Available Parameters:

Parameter Type Default Description
theme string 'light' Optional - Widget theme (light/dark/auto)
language string 'en' Optional - Widget language code
autoSync boolean true Optional - Enable automatic data synchronization
courseId string - Required - Your course identifier

API Examples

// Fetch course details fetch('https://teachmateai.com/api/v1/courses/123', { headers: { 'Authorization': 'Bearer YOUR_API_KEY' } }) .then(response => response.json()) .then(data => console.log(data)); // Create new course fetch('https://teachmateai.com/api/v1/courses', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ title: 'My Course', description: 'Course description' }) });

Example API calls using JavaScript fetch. Remember to replace YOUR_API_KEY with your actual API key.
